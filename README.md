## ActiveMQ介绍

## 对象模型
* ActiveMQConnectionFactory(连接工厂)
* ActiveMQConnection(连接对象)
* ActiveMQSession(会话对象)
* ActiveMQDestination(消息发送目的地)
* ActiveMQMessageProducer(消息生产者)
* ActiveMQMessageConsumer(消息消费者)
* ActiveMQMessage(消息对象)
